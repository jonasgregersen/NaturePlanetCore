Trin	Kommando	Forklaring
1	git checkout daniel	Gå til din branch
	git add . && git commit -m "msg"	Gem alt
2	git fetch origin	Hent nyeste fra GitHub
	git checkout main	Skift til main
	git pull origin main	Opdater main lokalt
3	git checkout main → git merge daniel	Merge dine ændringer ind i main
4	git push origin main	Push opdateret main til GitHub
5 (valgfrit)	git checkout daniel → git merge main	Hold din egen branch opdateret
